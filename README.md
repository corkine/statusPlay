# statusPlay

一个基于 Play Scala 的个人信息统计网站，内部使用 H2 嵌入式数据库。

- 网站服务可用性：通过 wsClient 进行网页可用性检查，默认 30min 检查间隔。
- 个人健康记录：通过 Apple Watch 检测每小时的身体活动：步数、静息和运动心率、卡路里、楼层、最大耗氧、心率变异、体重等。
